# Blink Frontend (Vue 3)

Vue 3 + Pinia client aligned to FastAPI endpoints.

## Run locally

```bash
cd frontend
npm install
npm run dev
```

By default, API base points to:
- `http://localhost:8787`

Override with env:

```bash
VITE_API_BASE=http://localhost:8787
```

## Legacy UI

Old static prototype is archived in:
- `frontend/legacy_static/`
